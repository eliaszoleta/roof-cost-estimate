# Roof Cost Estimator

A full-stack SEO-optimized roofing cost estimator SaaS with embeddable widget and roofer dashboard.

## Stack
- Frontend: React (CRA)
- Backend: Node.js / Express
- Database: Supabase
- Payments: Stripe
- Deploy: Railway (backend) + Vercel (frontend)
